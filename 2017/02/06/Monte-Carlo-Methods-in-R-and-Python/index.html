Monte Carlo Methods in R & Python

发表于 2017-02-06 | 分类于 随便写写

今天黄丹璐给我转了条微博，关于求不规则图形面积的，其中一个用到了Monte Carlo Methods.
然后我感觉学以致用装逼的时候到了…
就用r和python都写了写，反正没事练练也好，第一次完完全全自己写python，果然自己写比看各种tutorial进步要快得多。

第一个完整的自己写的可以在终端直接运行的python程序。
这一段是为了让黄丹璐能在自己电脑上也玩一玩。

在Macbook上运行python程序

  1. 打开Terminal (右上角🔍spotlight输入terminal）。
  2. 在$后面输入(前提是把MonteCarlo.py这个文件放在桌面上了） 摁Enter。
  3. 在$后面输入 摁Enter。
  4. 待出现”The Number of Samples:”，输入你想生成的样本次数（eg. 10000)，理论上数字越大越接近真实值，当然不要输太大，1,000,000搞得电脑宕机就不关我事了。
  5. 程序就会输出估计值，和弹出图片，点击图片窗口左上角关闭图片窗口，Terminal又会出现”The Number of Samples:”字样，可以重复第四步。

建议输入100，1000，10000，可以看到数字越大越接近真实值。
